# Problem

Given an array arr, we want to return all possible unique triplets that could sum up to 0(we can fix a target if we would want to).

## examples:

### input

-   arr = [-4, -2, 1, -5, -4, -4, 4, -2, 0, 4, 0, -2, 3, 1, -5, 0]

### output

-   [-5 1 4]
-   [-4 0 4]
-   [-4 1 3]
-   [-2 -2 4]
-   [-2 1 1]
-   [0 0 0]
